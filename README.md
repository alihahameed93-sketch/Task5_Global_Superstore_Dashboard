# Task 5: Global Superstore Business Dashboard

## Overview
This project focuses on building a lightweight business intelligence
dashboard using the Global Superstore dataset. The goal is to analyze
sales, profit, and customer performance and present key insights
through KPIs and visualizations.

The dashboard is implemented in Google Colab using Python, Pandas,
Matplotlib, and Seaborn for fast execution and clarity.

## Dataset
- Dataset: Global Superstore
- Key Columns Used:
  - Region
  - Category
  - Sub-Category
  - Customer Name
  - Sales
  - Profit

## Objective
- Analyze sales and profit performance
- Identify top-performing customers
- Provide clear business insights using visualizations

## Approach
1. Loaded and cleaned the Global Superstore dataset
2. Selected relevant business-related columns
3. Applied filters for Region, Category, and Sub-Category
4. Calculated key performance indicators (KPIs):
   - Total Sales
   - Total Profit
   - Top 5 Customers by Sales
5. Visualized top customers using bar charts
6. Summarized insights and conclusions

## Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Top 5 Customers by Sales

## Visualizations
- Bar chart displaying Top 5 Customers by Sales

## Skills Gained
- Data cleaning and preprocessing
- Business KPI analysis
- Data visualization with Matplotlib and Seaborn
- Business intelligence and data storytelling

## Conclusion
This project demonstrates how data can be transformed into meaningful
business insights through KPI analysis and visualization. The dashboard
helps stakeholders understand sales performance and identify high-value
customers, supporting data-driven decision-making.
